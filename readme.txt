== Instalation ==

=== Install from within WordPress ===

1. Visit the plugins page within your dashboard and select ‘Add New’;
2. Search for ‘Mobio for wooCommerce’;
3. Activate ‘Mobio for wooCommerce from your Plugins page;
4. Go to ‘after activation’ below.

=== Install manually ===

1. Upload the ‘mobio-for-woocommerce’ folder to the /wp-content/plugins/ directory;
2. Activate the ‘Mobio for wooCommerce plugin through the ‘Plugins’ menu in WordPress;
3. Go to ‘after activation’ below.

== Frequently Asked Questions ==

=== What is Mobio for WooCommerce?
Mobio for WooCommerce is a Payment Gateway that allows you to accept payments with SMS.

=== Is Mobio account required?
Yes! A Mobio account is required to receive the necessary credentials for Mobio.

=== How to get started?
Download, install and activate Mobio for WooCommerce from Plugins page in WordPress administration.
